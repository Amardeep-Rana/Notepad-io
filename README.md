# Notepad IO üìù

**Notepad IO** is a fast, complete offline, and privacy-focused notepad app built for everyone. Whether you're writing simple notes or looking at code, this app provides a clean and secure writing experience.

## üöÄ Built With
This app is built using **Flutter (Dart)**. It is designed to be lightweight and work on any Android and iOS device without the need for an internet connection.

---

## ‚ú® Key Features

- **Wide Format Support:** Open and edit many file types like `.java`, `.py`, `.js`, `.html`, `.json`, `.md`, `.srt`, and many more.
- **Secure PIN Lock:** Protect your private files with a 4-digit password (PIN) so only you can see them.
- **Find & Replace:** Easily search for specific words and replace them across your entire note.
- **Pinch-to-Zoom:** Change your font size quickly by using a pinch gesture (zoom in/out) on the screen.
- **No Permissions Needed:** This app does not require storage read or write permissions to function, keeping your device safe.
- **Save to Local:** Export your notes directly to any folder on your phone storage using Android's safe storage (SAF).
- **Modern Dark UI:** A beautiful dark interface for the app with a clean white canvas for focused writing.
- **Privacy First:** No ads, no tracking, and 100% offline. Your data stays on your phone only.
